# 🌿 遇己

> 小镇做题家的自我觉察工具 —— 遇见真实的自己

## ✨ 产品简介

遇己是一款专为「奥德赛时期」青年设计的心理健康自我觉察工具。

我们不灌鸡汤，不教你怎么往上爬。我们只是想帮你——
**听见自己内心的声音，看见真实的自己。**

## 🎯 目标用户

从小依靠应试教育完成阶层突围，县城/小镇原生家庭无资源托底，独自在大城市求学、打拼的青年。

长期被「分数=价值」单一标准驯化，成年后自动延伸出一套自我审判体系，终身活在"不够好"的内耗里。

## 🏗️ 三层产品架构

### 第一层：情绪客观观测
用数据打破自我PUA。
- 10秒快速心情记录
- 多维度情绪分类标签
- 30天情绪趋势可视化
- 分类情绪统计分析
- 所有数据本地存储（IndexedDB）

### 第二层：人生决策推演
听见内心的五个声音。
- 🧑‍⚖️ **卷王判官** — 世俗评价体系的声音
- 🦥 **躺平咸鱼** — 身体真实感受的声音
- 📊 **理性分析师** — 客观数据视角
- 🌱 **小孩初心** — 童年最纯粹的愿望
- 🕊️ **和解引导员** — 整合与觉察的声音

AI不会给你标准答案，只会帮你看清自己内心的真实倾向。

### 第三层：同频封闭社区
你不是性格有问题，你只是身边同类人太少。
- 📝 **焦虑拆解日记** — 匿名倾诉，同类拆解
- 📖 **选择实录** — 真实人生，没有标准答案
- ✨ **和解实验室** — 引导员带领的深度讨论

## 🛡️ 隐私优先

- **本地存储**：所有情绪日记、对话记录都存储在你本地的设备中
- **端侧AI**：支持WebLLM端侧推理，数据不上传云端
- **开源透明**：代码完全开源，任何人都可以审查数据流向
- **一键删除**：随时清除所有数据，不留痕迹

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript + Vite
- **样式方案**：TailwindCSS v4
- **路由管理**：React Router v6
- **本地数据库**：Dexie.js（IndexedDB 封装）
- **图表展示**：Recharts
- **图标库**：Lucide React
- **端侧AI**：WebLLM（@mlc-ai/web-llm）
