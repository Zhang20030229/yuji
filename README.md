# 🌿 遇己

> 奥德赛时期的自我觉察工具 —— 遇见真实的自己

## ✨ 产品简介

遇己是一款专为「奥德赛时期」青年设计的心理健康自我觉察工具。

我们不灌鸡汤，不教你怎么往上爬。我们只是想帮你——
**听见自己内心的声音，看见真实的自己。**

## 🎯 目标用户

正在经历人生迷茫期，探索自我、寻找方向的青年。
- 择业、定居、婚恋、人生意义……太多问题没有标准答案
- 长期被单一评价体系困住，活在"不够好"的内耗里
- 想要更了解自己，建立不依附外界的自我价值体系

## 🏗️ 三层产品架构

### 第一层：情绪客观观测
用数据打破自我PUA。
- 10秒快速心情记录
- 多维度情绪分类标签
- 30天情绪趋势可视化
- 分类情绪统计分析
- 本地存储 + 可选云端同步

### 第二层：人生决策推演
听见内心的五个声音。
- 🧑‍⚖️ **卷王判官** — 世俗评价体系的声音
- 🦥 **躺平咸鱼** — 身体真实感受的声音
- 📊 **理性分析师** — 客观数据视角
- 🌱 **小孩初心** — 童年最纯粹的愿望
- 🕊️ **和解引导员** — 整合与觉察的声音

AI不会给你标准答案，只会帮你看清自己内心的真实倾向。

### 第三层：同频封闭社区
你不是性格有问题，你只是身边同类人太少。
- 📝 **焦虑拆解日记** — 匿名倾诉，同类拆解
- 📖 **选择实录** — 真实人生，没有标准答案
- ✨ **和解实验室** — 引导员带领的深度讨论

### 第四层：年度觉察报告
长线觉察，年度复盘。
- 全年情绪数据统计
- 高频情绪标签分析
- 月度趋势对比
- 个性化洞察建议

## 🛡️ 隐私优先

- **本地存储**：所有数据默认存在浏览器本地
- **云端可选**：登录后可开启同步，数据跨设备无缝衔接
- **端侧AI**：WebLLM端侧推理，AI对话不经过任何服务器
- **开源透明**：代码完全开源，任何人都可以审查
- **一键删除**：随时清除所有数据，不留痕迹

## 🚀 快速开始

### 纯本地模式（无需配置）

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 云端同步模式（可选）

1. **创建 Supabase 项目**
   - 访问 https://supabase.com
   - 创建新项目

2. **配置数据库**
   - 在 Supabase SQL Editor 中运行 `supabase/schema.sql`

3. **配置环境变量**
   ```bash
   cp .env.example .env.local
   # 编辑 .env.local，填入你的 Supabase URL 和 Anon Key
   ```

4. **启动**
   ```bash
   npm install
   npm run dev
   ```

## 🛠️ 技术栈

| 类别 | 技术 |

| 本地数据库 | Dexie.js（IndexedDB 封装） |
| 云端数据库 | Supabase（可选） |
| 图表展示 | Recharts |
| 图标库 | Lucide React |
| 端侧AI | WebLLM（@mlc-ai/web-llm） |
